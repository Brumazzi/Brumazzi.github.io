Getting Started

What's Quartzo?

Quartzo is a CSS framework created to be simple and easy to use. Its components, for the most part, do not use javascript, allowing you to build complete layouts only with HTML and CSS.

How to import?

Download Quartzo framework and copy CSS and JS files to your project folder.

$ cd project_path
$ wget https://github.com/brumazzi/quartzo.css/archive/refs/tags/v1.3.0.zip
$ unzip quartzo-1.3.0.zip
$ cp quartzo-1.3.0/dist/css/quartzo.css css_folder_path
$ cp quartzo-1.3.0/dist/css/quartzo.js js_folder_path
