Ministyle

JS is not a solution, it's a problem! Use when there is no way out.

Ministyle is a StyleSheet library created to make writing code easier, removing complex struct to generate components and reducing your html.

Basic Functions

Spacing, Typographys, Displays, Utils

.rounded round of the borders.

.no-rounded remove round of the borders.

.shadow add border shadow.

.w-{{25|50|75|100}} define width to element.

.h-{{25|50|75|100}} define height to element.

.overflow-{{hidden|auto}} add or remove scrollbar.

.overflow-x-{{hidden|auto}} add or remove scrollbar in x axis.

.overflow-y-{{hidden|auto}} add or remove scrollbar in y axis.

.text-{{start|center|end|justify}} align text.

.line-height-{{0|1|2|3|4}} set line height to element.

.p-{{0|1|2|3|4|5}} set padding on element.

.px-{{0|1|2|3|4|5}} set padding on element to left and right.

.py-{{0|1|2|3|4|5}} set padding on element to top and bottom.

.p{l|t|r|b}-{{0|1|2|3|4|5}} set padding on element to one direction.

.m-{{0|1|2|3|4|5}} set margin on element.

.mx-{{0|1|2|3|4|5}} set margin on element to left and right.

.my-{{0|1|2|3|4|5}} set margin on element to top and bottom.

.m{l|t|r|b}-{{0|1|2|3|4|5}} set margin on element to one direction.

.p-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set padding on element using screen size.

.px-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set padding on element to left and right using screen size.

.py-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set padding on element to top and bottom using screen size.

.p{l|t|r|b}-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set padding on element to one direction using screen size.

.m-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set margin on element using screen size.

.mx-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set margin on element to left and right using screen size.

.my-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set margin on element to top and bottom using screen size.

.m{l|t|r|b}-{sm|md|ld|xl}-{{0|1|2|3|4|5}} set margin on element to one direction using screen size.

.d-{{none|block|inline|table|inline-block}} set layout on element.

.d-{sm|md|ld|xl}-{{none|block|inline|table|inline-block}} set layout on element using screen size.

.b add border on element.

.b-0 remove border from element.

.b{l|t|r|b} add border on element on one direction.

.b{l|t|r|b}-0 remove border from element on one direction.

.b-{sm|md|ld|xl} add border on element using screen size.

.b-{sm|md|ld|xl}-0 remove border from element using screen size.

.b{l|t|r|b}-{sm|md|ld|xl} add border on element on one direction using screen size.

.b{l|t|r|b}-{sm|md|ld|xl}-0 remove border from element on one direction using screen size.
